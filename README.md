# Beck Projects LLC Invoice App

Phone-friendly Vercel web app for quotes, invoices, job photos, material quote/receipt uploads, contractors, and simple tax reporting.

## Run locally
npm install
npm run dev

## Deploy to Vercel
1. Create a free account at vercel.com.
2. Create a new GitHub repository and upload these files, or use Vercel CLI.
3. Import the project in Vercel.
4. Framework: Vite. Build command: npm run build. Output directory: dist.
5. Deploy.

## Important
This first version stores data in your browser/device using localStorage. For business use across multiple phones/computers, the next version should add a cloud database, login, file storage, and backups.
